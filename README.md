💡 Questões Discursivas
7. 📚 Diferença entre Diagrama de Classes e de Objetos
Diagrama de Classes: mostra a estrutura estática do sistema, ou seja, as classes, atributos e métodos.

Diagrama de Objetos: mostra uma foto do sistema em execução, ou seja, instâncias reais de uma classe.

Exemplo prático:
Classe Pessoa (Diagrama de Classes)
Instância Pessoa1: nome = João, idade = 25 (Diagrama de Objetos)

8. 🧩 Elementos do Diagrama de Componentes
Identificador (ícone que indica ser um componente)

Agrupamentos de classes

Relacionamentos entre os módulos

Utilizado para mostrar organização física do software, como pacotes, bibliotecas e conexões.

Exemplo prático:
Em um sistema bancário, há componentes como:
🔹 Módulo de Autenticação
🔹 Módulo de Transferência
🔹 Módulo de Emissão de Extrato
